# Shapestest

A Game made with Funcode (a trash like game engine).
v0.2已更新，更少的bug更多的歌曲。


## Funcode Intro
基于MFC的游戏引擎，被各大高校广泛使用。


## Feature
- 原创的立绘和原创的剧情！跌宕起伏！
- 可扩展谱面与剧情系统！
- MathHandle的助力
- 街机式过关流程，只需花费你${money}元
